# nitrokey-sys-rs

Low-level Rust bindings for `libnitrokey`, providing access to Nitrokey
devices.

```toml
[dependencies]
nitrokey-sys = "0.1.0"
```

Currently, this crate requires a `libnitrokey` installation available in the
default library and include path.
